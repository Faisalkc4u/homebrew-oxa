# Oxa CLI - Homebrew Installation

Oxa is a powerful CLI tool for Mac. Install it easily using Homebrew!

## Installation

First, add the Oxa tap:

```sh
brew tap Faisalkc4u/oxa
```

Then, install Oxa:

```sh
brew install oxa
```

## Upgrade

To update to the latest version:

```sh
brew upgrade oxa
```

## Uninstall

If you need to remove Oxa:

```sh
brew uninstall oxa
```
